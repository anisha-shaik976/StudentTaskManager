# Student Task Manager

## Project Overview
Student Task Manager is an Android application developed using Kotlin and Android Studio.

## Features
- Login Authentication
- Welcome Admin Message
- Student Data Retrieval
- API Integration using Retrofit
- Loading Indicator
- Error Handling
- Network Failure Handling

## Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- Retrofit
- REST API
